Question 5 (Extra): Which states are the most dangerous if you take population into consideration?


One could say that the number of incidents in a state defines how threatening or dangerous a state is. On the other hand, the number of deaths per shooting might define the real danger of a state. Considering the population of a state, every graph is plotted per capita. Graph one shows the number of gun violence incidents per capita for every state, while graph two shows the number of people killed per incident for every state.
